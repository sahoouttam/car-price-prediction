# 🚗 Car Price Prediction API

An end-to-end Machine Learning project that predicts the selling price of a used car based on its features.

This project includes:

- Data preprocessing pipeline
- Multiple regression models
- Hyperparameter tuning
- Model serialization
- Production-ready API using FastAPI

---

## 📌 Features

✔ Data Cleaning & Missing Value Handling  
✔ Feature Engineering (Car Age calculation)  
✔ Categorical Encoding (OneHotEncoder)  
✔ Model Training (Linear, Decision Tree, Random Forest)  
✔ Hyperparameter Tuning (GridSearchCV)  
✔ Model Evaluation (R², MAE, MSE, RMSE)  
✔ Saved Trained Model (`joblib`)  
✔ REST API using FastAPI
